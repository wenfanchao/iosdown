# 第一次使用git命令提交代码
1. ##### 在新文件夹中使用 git init 命令
```
$ sudo git init
```
2. ##### 创建 README.md 文件
```
$ sudo touch README.md
```
3. ##### 使用 git commit 命令
```
$ sudo git commit -m "提交描述"
```
4. ##### 使用 git remote 增加一个新的远程仓库，并命名
```  
$ sudo git remote add [仓库名称] [仓库地址]
```
5. ##### 取回远程仓库的变化，并与本地分支合并
```
$ sudo git pull [仓库名称] [分支名称]
```
6. ##### 上传本地指定分支到远程仓库
```
$ sudo git push [仓库名称] [分支名称]
```
